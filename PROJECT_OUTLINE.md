### 🟡 中优先级 — 丰富度提升

#### 6. 领地系统（20级解锁）
```
购买条件：20级，500银币

功能模块：
- 农田 Lv1（3块田地）：种植作物
- 矿场 Lv1（1个矿洞）：采集矿石
- 作坊 Lv1（2个制作位）：制作道具
- 花园 Lv1（15级解锁）：种植特殊材料

收入：离线收益/每日收获
```

#### 7. 图鉴系统
```
消耗未绑定装备激活图鉴：
- 激活后永久获得基础属性+10%
- 装备图鉴（武器/副手/头饰/衣服/腰带/鞋子/佩戴）
- 43种装备图鉴

前端入口：状态页 → 图鉴Tab
```

#### 8. 装备套装效果
```
同套装多件激活额外属性：
- 2件套：防御+5%
- 3件套：攻击+10%
- 4件套：HP上限+15%

沧澜套装示例：
- 银月星光套装（月卡送）
- 海盗王套装
```

#### 9. 称号/成就系统
```
称号获取方式：
- 等级称号（初出茅庐/小有名气等）
- 成就称号（完成特定成就解锁）
- 限时称号（活动获得）

属性加成：称号带属性
```

#### 10. 船舶强化/改造
```
当前只有买船/卖船，需增加：
- 船帆升级（提升航速）
- 船身加固（提升HP上限）
- 特殊装备（撞角/旗帜/导航仪）
- 船只外观/皮肤
```

#### 11. 航海图鉴
```
记录已发现的：
- 城市（解锁港口信息）
- 地点（探索度）
- 怪物图鉴（击杀计数/掉落统计）
```

#### 12. NPC对话扩展
```
当前npc_dialog只有33条，需大量补充：
- 每个城市NPC 10+ 条对话
- 任务相关对话（接任务/交任务/进行中）
- 随机闲聊（不同时间段不同内容）
```
