## 五、技术架构

```
客户端: Vue3 + Vite + Pinia + VueRouter
服务端: Node.js + Express
数据库: MySQL
实时通信: WebSocket
```

### 服务地址
- HTTP API: http://localhost:3000
- 前端: http://localhost:5173
- WebSocket: ws://localhost:8282

### 关键表结构
- `user` - 玩家数据 (含 ship_id, mp/mp_max, sail_* 航海状态, buff_end/buff_flags 增益状态)
- `place` - 地点(4956)
- `item` - 装备/物品 (含 quality 品质字段)
- `inventory` - 背包物品 (含 durability/durability_max 耐久度)
- `item_affix` - 词缀定义表
- `inventory_affix` - 背包物品词缀表
- `skill` - 技能定义表
- `user_skill` - 用户技能表 (含 cooldown_end)
- `status_effect` - 状态效果定义表 (17种状态)
- `user.status_effects` - 用户当前状态效果 (JSON)
- `monster` - 怪物
- `quest` - 任务
- `npc` - NPC
- `ship` - 船只 (含 hp/hp_max)
- `user_ship` - 用户船只 (含 hp/hp_max 船只状态)
- `cdkey` / `cdkey_reward` / `cdkey_log` - CDKEY系统
- `goods` / `market_price` / `cargo` - 贸易系统
- `npc_shop_item` - 商店商品
- `monster_drop` - 掉落表

### 新增 API 端点
| 端点 | 方法 | 说明 |
|------|------|------|
| `/api/ship/status` | GET | 获取船只HP状态 |
| `/api/ship/repair` | POST | 码头修理船只 |
| `/api/item/repair` | POST | 铁匠修理装备 |
| `/api/npc/skills/all` | GET | 获取所有技能 |
| `/api/npc/skills/my` | GET | 获取已学技能 |
| `/api/npc/learn-skill` | POST | 向NPC学习技能 |
| `/api/user/status` | GET | 查询当前Buff/Debuff状态 |
