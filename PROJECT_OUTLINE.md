## 三、缺失功能（按优先级）

### 🔴 高优先级 — 核心体验缺口

#### 1. 每日活跃系统
参考沧澜四海设计：
```
活跃度任务（每日0点重置）：
- 怪物猎手（20活跃）：击杀任意怪物 0/100
- 副本先锋（30活跃）：完成任意副本 0/1
- 竞技新星（20活跃）：参与竞技场挑战 0/1
- 钓鱼达人（15活跃）：成功钓鱼 0/3
- 任务达人（15活跃）：完成任意任务 0/1
- 市场贸易（20活跃）：城市间贸易5次 0/5

活跃度宝箱（20/40/60/80/100档）：
- 20：千银矿石×1
- 40：体力宝×2
- 60：龙泉水×2
- 80：月华密令×1
- 100：龙门镖旗×2
```

#### 2. 在线奖励
```
每5分钟可领取一次：
- 在线0分钟 → 还需5分钟
- 累计在线时长奖励（银币/体力宝/材料）
```

#### 3. 城内地图系统

码头 | 教堂 | 东城门 | 珠宝店 | 市场 | 商店
花园 | 王宫 | 警察局 | 住宅区 | 研究院 | 南城门
劳动广场 | 工厂区 | 旧城区 | 监狱

需要：place表补充城市内部地点，完善n/s/e/w导航

### 🟡 中优先级 — 丰富度提升

#### 8. 领地系统（20级解锁）
```
购买条件：20级，500银币

功能模块：
- 农田 Lv1（3块田地）：种植作物
- 矿场 Lv1（1个矿洞）：采集矿石
- 作坊 Lv1（2个制作位）：制作道具
- 花园 Lv1（15级解锁）：种植特殊材料

收入：离线收益/每日收获
```

#### 9. 图鉴系统
```
消耗未绑定装备激活图鉴：
- 激活后永久获得基础属性+10%
- 装备图鉴（武器/副手/头饰/衣服/腰带/鞋子/佩戴）
- 43种装备图鉴

前端入口：状态页 → 图鉴Tab
```

#### 10. 装备套装效果
```
同套装多件激活额外属性：
- 2件套：防御+5%
- 3件套：攻击+10%
- 4件套：HP上限+15%

沧澜套装示例：
- 银月星光套装（月卡送）
- 海盗王套装
```

#### 11. 称号/成就系统
```
称号获取方式：
- 等级称号（初出茅庐/小有名气等）
- 成就称号（完成特定成就解锁）
- 限时称号（活动获得）

属性加成：称号带属性
```

#### 12. 船舶强化/改造
```
当前只有买船/卖船，需增加：
- 船帆升级（提升航速）
- 船身加固（提升HP上限）
- 特殊装备（撞角/旗帜/导航仪）
- 船只外观/皮肤
```

#### 13. 航海图鉴
```
记录已发现的：
- 城市（解锁港口信息）
- 地点（探索度）
- 怪物图鉴（击杀计数/掉落统计）
```

#### 14. NPC对话扩展
```
当前npc_dialog只有33条，需大量补充：
- 每个城市NPC 10+ 条对话
- 任务相关对话（接任务/交任务/进行中）
- 随机闲聊（不同时间段不同内容）
```

```
月卡：
- 每日赠送体力宝×5

```
邀请码：每个玩家唯一邀请码

```
摆摊系统：
- 玩家在市场页摆摊
- 上架物品+定价
- 下架/修改价格

市场税率：每次交易收取5%税
```

#### 19. 声望/阵营系统
```
阵营：地中海/北海/东亚 等

组队航行：

```
公会战：
- 宣战→攻城→守城

```

```
每日跑商：

- 低买高卖特定商品
- 完成后获得商贡和银币
```

#### 23. 宠物进化/变异
```
进化路径：
- 等级提升→进化（外形变化）
- 变异：随机属性变化
- 技能学习/遗忘
```

#### 24. 宝石系统
```
打孔：装备打孔（最多3孔）
镶嵌：宝石镶嵌增加属性
宝石类型：攻击/防御/生命/敏捷/暴击

#### 25. 婚姻系统

求婚→订婚→结婚

#### 26. 卡片/圣痕系统

#### 27. 世界BOSS
