## 四、技术架构

```
客户端: Vue3 + Vite + Pinia + VueRouter
服务端: Node.js + Express
数据库: MySQL
实时通信: WebSocket
```

### 服务地址
- HTTP API: http://localhost:3000
- 前端: http://localhost:5173
- WebSocket: ws://localhost:8282

### 关键表结构
- `user` - 玩家数据
- `place` - 地点(4956)
- `item` - 装备/物品
- `monster` - 怪物
- `quest` - 任务
- `npc` - NPC
- `ship` - 船只
- `cdkey` / `cdkey_reward` / `cdkey_log` - CDKEY系统
- `goods` / `market_price` / `cargo` - 贸易系统
- `npc_shop_item` - 商店商品
- `monster_drop` - 掉落表
