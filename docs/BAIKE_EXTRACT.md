## 一、游戏背景

大航海时代，玩家化身冒险者，组建船队、成立帮派、攻城略地。
挑战：海盗、巨浪、海中生物、森林巨兽、古堡吸血鬼、沙漠木乃伊。
剧情关键词：艾丽莎、神秘鳞片、马可酒馆、威尼斯。
特色文化：希腊神话、北海女巫、非洲大陆、古代中国、地中海、蓬莱仙岛。

## 二、任务类型（6种）

1. **找人**：找到相关NPC对话
2. **杀怪**：杀死指定数量怪物后复命
3. **寻物**：收集物品（杀怪掉落/商店购买）
4. **杀BOSS**：击败指定BOSS，难度高体力多
5. **运货**：市场购买商品交给相关NPC
6. **回答问题**：答对NPC问题

## 四、强化系统（强化共15级）

1. 强化装备增加**体力、攻、防、敏**属性
2. 强化需要**金或银贝+龙泉水**，等级越高消耗越多
3. 强化共分**15级**：
   - 每升1级附加**体力属性**
   - 第3级开始增加**攻防敏属性**
   - 第10级强化后：**体力值=道具等级×10**，攻防敏属性增加**×20%**
4. 强化等级颜色标识（4~20级）：
   ```
   红 → 橙 → 黄 → 绿 → 青 → 蓝 → 紫 → 粉 → 幻 → 天 → 弑 → 神 → 灵 → 真 → 玄 → 尊
   ```
   （尊色为最高级装备）
5. **强化失败**：只损失部分原料，NPC返还部分原料，**装备不会有损失或降级**

## 八、帮会与攻城

### 攻城规则
1. 必须由**帮主/帮副**发起
2. 集合时间：**300秒**
3. 攻城时间：**12:00~14:00** 和 **20:00~22:00**
4. 帮会人数>20人、在线>10人才可宣战
5. 宣战费用：**10银**
6. 胜利条件：击败所有敌人
7. 30分钟未攻下→**守方赢**
8. 城市占领后**1小时**不能再次攻击
9. 帮会收入：城市**税收**、帮会战争、玩家**捐献**

### PK规则
1. 帮会内成员**不允许PK**
2. 猎人大厅可领**通缉令**
3. 同帮会人员**不可通缉**
4. 可通过攻城、掠夺获取财富

## 九、贸易系统

1. 不同城市**交易市场**买卖货物（赚差价）

4. 装备道具药品买卖
5. 结婚/师徒/帮会捐赠
6. 攻城守城掠夺
