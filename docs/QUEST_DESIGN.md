### 任务1：清理城郊野狗

**接取NPC**：马可（Marco，酒馆老板）  
**任务类型**：杀怪  
**目标**：击败野狗×3  
**等级需求**：Lv1  
**前置任务**：无

**任务描述**：
> *「城郊最近野狗成群，已经咬伤了好几个路人。你看起来有点身手，帮我清理掉那几只野狗吧。」*—— 马可

**NPC对话（接取）**：
```
马可：又有个年轻人来酒馆了。你看起来身体恢复得不错……对了，城郊最近有野狗出没，
     已经伤了好几个人了。你能帮我清理掉3只吗？就在城北门外不远。
```

**NPC对话（完成）**：
```
马可：干得漂亮！这些野狗终于安静了。这点钱和药水你拿着，算是辛苦费。
     对了，你接下来有什么打算？
     要不……去码头找我聊聊？威尼斯虽小，但也是通往世界的大门。
```

**奖励**：EXP+100，铜币+200，小回复药×2

**后续引导**：自动接取任务2「初识航海」

### 任务2：初识航海

**接取NPC**：马可（威尼斯码头）  
**任务类型**：对话  
**目标**：与马可对话  
**等级需求**：Lv5  
**前置任务**：任务1「清理城郊野狗」

**任务描述**：
> *「威尼斯码头是地中海最繁忙的港口之一。去码头找马可，听听他关于航海的建议。」*

**NPC对话（接取）**：
```
马可：想出海？好，有志气！不过出海可不比打狗，你得先有艘自己的船。
     威尼斯码头的船坞有卖小帆船，价格不贵。当然，你也可以先到处的跑腿，
     熟悉熟悉这个世界，攒够了钱再说。
```

**NPC对话（完成）**：
```
马可：航海之路充满未知，但也充满机遇。你已经做好了心理准备吗？
     我听说雅典最近有些商人在找帮手，如果你想赚钱，不妨去那边看看。
     去雅典的船票不贵，码头直接买就行。
```

**奖励**：EXP+120，铜币+300，**小帆船×1**（可选）

**后续引导**：自动接取任务3「雅典之行」

### 任务3：雅典之行

**接取NPC**：马可  
**任务类型**：到达（前往place=1011雅典）  
**目标**：抵达雅典城  
**等级需求**：Lv10  
**前置任务**：任务2「初识航海」

**任务描述**：
> *「从威尼斯码头起航，前往希腊的雅典城。那里的商人或许能提供新的机遇。」*

**航行路线**：威尼斯（地中海）→ 雅典（地中海）

**NPC对话（接取）**：
```
马可：雅典是地中海东部的重要港口，那里商贾云集。你去那边找当地的商人打听打听，

```

```
     最近正好有件事需要人帮忙——我的一批货物被海盗盯上了，你能帮我查查是怎么回事吗？
```

**奖励**：EXP+200，铜币+500

**后续引导**：自动接取任务4「雅典商人」

**接取NPC**：法拉  
**任务类型**：对话  
**目标**：返回威尼斯与马可对话  
**等级需求**：Lv22

**航行路线**：亚历山大 → 威尼斯

**NPC对话（接取）**：
```
法拉：去找威尼斯的马可。告诉他："龙涎醒了"。

```

```
     那枚鳞片……来自一个叫做龙涎的远古存在。你之所以被漂流到威尼斯海边，
     不是因为海盗——是因为龙涎之门在呼唤你。
     你必须继续往东走，去长安。那里有解开一切的钥匙。
```

**奖励**：EXP+300，铜币+500，**龙涎之门地图×1**（任务道具，显示隐藏坐标）

**目标**：击败精英匪霸（第8层）×1  
**等级需求**：Lv40

> *「深入牛头山第8层，找到精英匪霸，击败他！」*

```

     但以你现在的水平，应该能应付。
```

**NPC对话（完成）**：
