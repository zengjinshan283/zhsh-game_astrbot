## 二、角色成长

### 等级系统
- 初始等级：1级
- 经验来源：任务/打怪/日常活跃
- 每级所需经验：约 `level * 100`（待调）

### 战斗属性

| 属性 | 说明 |
|------|------|
| HP | 生命值，死亡则进入复活流程 |
| MP | 魔法值，施放技能消耗 |
| ATK | 攻击力，影响伤害计算 |
| DEF | 防御力，减免伤害 |
| AGI | 敏捷，影响先手与闪避概率 |
| CRIT | 暴击率（%），暴击造成双倍伤害 |

## 四、战斗系统

### 回合制自动战斗

```
每回合：
1. 双方各自根据 AGI 排序先手
2. 玩家回合：可普攻 / 施放技能 / 使用道具 / 逃跑
3. 敌人回合：普攻或技能
4. 结算 HP/MP，检查死亡/胜利
```

### 伤害公式

```
最终伤害 = ATK - DEF/2 （最低为 ATK 的 10%）
暴击：最终伤害 × 2
```

### 航海战斗（船只战）

- 船只拥有独立 HP（`user_ship.hp`）
- 船只受损会影响航行安全（待实现）
- 船只可修理（消耗铜币）

### 副本战斗

- 牛头山：12层，怪物 Lv30-50，产出装备强化材料
- 四象圣殿：4宫（白虎/玄武/青龙/朱雀），各4层，怪物 Lv100

## 五、装备系统

### 品级

| 品级 | 颜色 | 词缀数 | 说明 |
|------|------|--------|------|
| 白 | 白色 | 0 | 无词缀，基础属性 |
| 绿 | 绿色 | 1 | 1条绿色词缀 |
| 蓝 | 蓝色 | 2 | 2条蓝色词缀 |
| 紫 | 紫色 | 3 | 3条紫色词缀 |
| 橙 | 橙色 | 4 | 4条橙色词缀 + 特殊效果 |

### 词缀池（item_affix）

| 词缀类型 | 属性 |
|----------|------|
| 攻击强化/提升/增强 | ATK + |
| 防御强化/提升/增强 | DEF + |
| 生命强化/提升/增强 | HP + |
| 敏捷提升/增强 | AGI + |
| 暴击强化/增强 | CRIT + |
| 生命偷取 | 造成伤害时 % 转化为 HP |
| 金币加成 | 战斗获得额外金币 |

### 耐久度

- 装备有 `durability` / `durability_max`
- 每次战斗后降低（待定）
- 耐久归零：装备效果消失，可修理由铁匠

### 技能列表（初版）

| 技能 | 类型 | 倍率 | MP消耗 | 冷却 | 解锁等级 |
|------|------|------|--------|------|----------|
| 重击 | 主动 | ATK×1.5 | 10 | 30s | Lv5 |

| 强力一击 | 主动 | ATK×2.0 | 25 | 60s | Lv15 |

### 技能类型

- **主动技能**：战斗中手动施放，消耗 MP，有冷却
- **被动技能**：永久生效，不占用战斗行动

## 十二、成长规划建议

### 阶段一（Lv1-30）

- 主线推进为主
- 熟悉战斗/装备/技能系统
- 驯兽师支线获取宠物

### 阶段二（Lv30-50）

- 牛头山副本循环
- 城市循环支线（赚铜币/练级）
- 装备强化积累

### 阶段三（Lv50-80）

- 四象圣殿挑战
- 公会加入/社交
- 高级技能解锁

### 阶段四（Lv80-100）

- 四象圣殿全宫通关
- 橙色装备追求
- 四海之王结局
